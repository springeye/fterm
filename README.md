# fterm

fterm是一款基于Flutter开发的跨平台终端工具

# 功能
- [x] 本地终端
- [x] 多种主题颜色
- [x] ssh主机连接
- [x] ssh配置本地加密存储
- [x] 通过ssh跳板机连接目标主机
- [x] 支持zmodem
- [ ] ssh配置多端同步
